# VSRegex

[ChinaMobilePhoneNumberRegex] wrappers for iOS and macOS in Swift.

## Features
- Pure-Swift interface
- Well-documented
- Extensively tested

## Requirements

- iOS 8.0+ / macOS X 10.9+ / tvOS 9.0+ / watchOS 2.0+
- Xcode 9.0+
- Swift 4.0+

## Quick Start

```swift
import VSRegex

let isMatch = VSRegex.matches("+8613800138000")
print(isMatch) // true
```

## Installation

### CocoaPods

```
pod 'VSRegex', '~> 1.0.0'
```
